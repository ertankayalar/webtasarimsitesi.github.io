Web Tasarım Firmaları Araştırma ve Seçme Rehberi

Yazan Ertan Kayalar | Güncelleme Mar 1, 2019 | Genel |

Acemiler için Web Tasarım Firmaları Nasıl Araştırmalı ve Nasıl Değerlendirilmeli

Web tasarım firmalarını adım adım inceleme

Hangi kriterlere dikkat etmelisiniz ?

Kaçınmanız gereken riskler nelerdir ?

Yazan: Ertan Kayalar

18 yıldır bu web tasarım sektöründe çalışıyorum. Bulunduğum çevrede web tasarım firması kuran ilk kişi olarak yıllardır müşteri talepleri ile karşılaşıyorum. Müşterilerimde en sık karşılaştığım durum web tasarım firmaları arasından seçerken nasıl yapacaklarını bilmiyorlar.

Günümüzde insanlar zamanlarının çoğunu (TV’den daha çok)  internette geçiriyor. Satın alma kararlarını web sitelerine bakarak karar veriyorlar.

Fakat pek çok şirketin web sitesi yeterli düzeyde değil.
Bu noktada hatalı karar veren şirketler internet fırsatlarını kaçırıyorlar. Tek kelimeyle özetlersek, para kaybediyorlar.
Bu rehberi, web tasarım firmaları arasından en doğru tercihi nasıl yapmanıza yardımcı olması için hazırladım.
Web tasarım firması araştırması üç adımdan oluşuyor:
  1. Hazırlık
  2. Web Tasarım Firması Araştırması
  3. Değerlendirme

Adım 1: Hazırlık Aşaması: Web Tasarım Projeniz üzerinde düşünün

 

Web tasarım firmaları araştırmanızdan önce kendinize sormanız gereken sorular var. Web sitenizin ihtiyaçlarını iyi belirlemeniz gerekiyor. 

Malesef, ihtiyaçlar belirlenmeden hemen fiyat araştırması işine geçiliyor… 

1) Doğrudan fiyat sorma işine dalmayın

Yıllardır aynı sorunla uğraşıyoruz:  “Ertan bey, web sitesimiz yarım kaldı, nasıl toparlarız ?”

 

Senaryo aşağı yukarı şöyle işliyor:
  • Web sitesi fikri birkaç cümle ile açıklanır, fiyat sorulur.
  • En düşük fiyatı veren web tasarım firması projeyi alır.
  • Başta konuşulmayan noktalar sonradan ortaya çıkar.
  • Fiyat, süre başta olmak üzere önemli değişiklikler olur.
  • Yeniden değerlendirme daha yüksek bir fiyatla devam eder veya…
  • Yarım kalan proje (genellikle en başa dönerek) yeni web tasarım firması aranır.
Web sitesi sahibi, ucuz web tasarımı ararken kendisini kandırır.

 

Benim önerim:
Fiyat aşamasına gelmeden önce isteklerinizi, proje detaylarınızı iyi belirleyin.

2) Proje detaylarınızı ve hedeflerinizi belirleyin

Herşeyden önce nasıl bir site yaptırmak istiyorsunuz ? Neleri hedefliyorsunuz ? Hedef olarak “web sitesini açmak” cevabını verenler oluyor. Ancak, hedefi olmayan bir gemiye hiç bir rüzgar yardım edemez.

 

İşte size örnek hedefler:
  • Gelir elde etmek
  • Üye toplamak
  • Daha çok ziyaretçi çekmek
  • Şirketten şirkete satışlarınızı arttırmak
  • Web sitenizi mobil cihazlarla uyumlu hale getirmek
  • Eposta abone sayınızı arttırmak
Liste uzar gider. Hedefleriniz size özeldir. Bir hedef belirlemeden web sitesi yaptırmak boşuna bir çabadır. Vereceğiniz her para ve zaman boşuna akan nehir gibidir.

 

Hedeflerinizi belirledikten sonra, web sitenizin detaylarını belirlemeye geçebilirsiniz.
Projenizin detaylarını belirlemenin faydaları:
  • Web sitenizin ihtiyaçlarını ortaya çıkarmak.
  • Bu ihtiyaçlara göre web tasarım firmasında arayacağınız yetenekleri ve teknik özellikleri belirlemek.
  • Sizin için en iyi web tasarım firmasını daha kolay seçmek.

 

Bir fiyat arayışı ile başlamak yerine aşağıdaki konuları düşünün:
  • Projenizin hedefi nedir ? Ne yapmak istiyorsunuz ?
  • Web sitenizin ana fikri nedir ? Kurumsal tanıtım, online mağaza, blog, haber portalı, sosyal medya vs
  • Web sitenize konacak içerik hakkında bir tahminde bulunun. Kaç sayfa, ürün, resim vb. Bu bilgilerin yerleştirilmesini web tasarım firmasından mı istiyorsunuz, kendiniz mi yapacaksınız ?
  • Web sitenizde istediklerinizi maddeleştirin.
  • Özel talepleriniz varsa bunları multaka yazın. Web sitenizin ihtiyaçlarını belirleyecek kadar detay yeterlidir.
  • Web hosting, alan adı satın alımı yaptınız mı ? Bu hizmetler için web tasarım firması mı size yardımcı olacak ?
  • Ne kadar sürede tamamlanmasını istiyorsunuz ? Aciliyetiniz nedir ? Gerekmedikçe kısa süreler belirlemeyin.
  • Bu işe ne kadar bütçe ayırıyorsunuz ?
  • Pazarlamaya ihtiyacınız olacak mı ?
Detaylara göre web tasarım firmasında aradığınız yeteneklerin de farkına varmış olacaksınız.

 

Örneğin:
  • İçerikleri sık sık değiştirecekseniz WordPress gibi programlara destek veren bir firmayı arıyorsunuz demektir.
  • Eğer web sitenizdeki ürün veya hizmetlerin pazarlanması gerekiyorsa, web tasarım firmasında dijital pazarlama hizmetlerini de aramanız gerekecek.
  • Tamamen özel bir proje yaptırıyorsanız yazılım uzmanı veya yazılım geliştirme hizmeti arıyorsunuz demektir.

“Ne çok detay var ?” diye düşünmeyin. Kendi anlayacağınız seviyede detayları belirlemek yeterlidir.

3) Gerçek probleminizin içerik olduğunun farkına varın

 

Web tasarımı amaç değil araçtır.

 

Web sitenizin hedef kitlenizle bağlantı kurması ancak içerikle gerçekleşir.

 

Arama motorlarında ilk sıralarda çıkmak için, sosyal medyada popüler olmak için, hedeflerinize ulaşmak için içerik birinci ve esas faktördür.

Şu soruyu kendinize her zaman sormayı unutmayın: web sitenize insanlar neden gelecek ?

Kendinizi onların yerine koyun.

Adım 2: Web Tasarım Firmaları Araştırması

Hazırlık aşamasında nasıl bir web sitesi istediğinizi belirlediniz. Şimdi sıra en uygun web tasarım firmasını belirlemeye geldi.
Web Tasarım Firmalarını Nerede Bulacağım ?
Elbette ilk akla gelen kaynaklar Google veya Yandex gibi arama motorları.
Ancak, Linkedin gibi profesyoneller ağını da tavsiye ederim. Linkedin içerisindeki şirket sayfalarını da iyi bir araştırma kaynağıdır. Ayırca Facebook ve Pinterest gibi sosyal medyaları da araştırabilirsin.
Araştırmanın kritik noktasını web tasarım firmalarının  web siteleri oluşturacak.

Web Tasarım Firmaları Araştırma Kriterleri

Eğer profesyonel bir web tasarımı istiyorsanız, doğru web tasarım firmasını seçmek zorundasınız.
İşte size en uygun tercihi yapmak için kolaylık sağlayacak tavsiyeler.
1) Web sitelerinin durumunu inceleyin

İlk işiniz özenle hazırlanmış bir web siteleri var mı kontrol etmek olsun.

  • Web sitesinin görünümü nasıl ?
  • Mobil cihazlarda kolayca kullanılabiliyor mu ?
  • Bilgileri güncel mi ?
  • En son blog sayfasına ne zaman yazı paylaşmışlar ?
Web siteleri ile düzenli ilgilenmeleri önemli. Çünkü, size yapacakları iş konusunda da ne kadar titiz olacaklarını tahim edebilirsiniz.
Basmakalıp “bir numarayız” türünde reklam kokan lafların bir yararı olmadığını biliyorsunuz. Pek çok tasarımcının web sitesinin bile olmadığını görünce şaşırmıyor musunuz ?

Kendi web sitesi olmayan, “tasarım aşamasında” lafları olan firmaları eleyin.

 

İyi bir web tasarım firması öncelikle “aslan yatağından belli olur” sözüyle, web sitesine gerekli özeni göstermek zorundadır.
“Vakit yok” palavralarına inanmayın.
Bu işini gerçekten seven kendi sitesinde neler yapmaz ?
2) Bilgilerini, deneyimlerini paylaşıyorlar mı ?
İnternet bilgi paylaşımının kazanca dönüştürüldüğü bir dünyadır.
Başarılı web tasarım firmaları müşterilerin ihtiyaçlarına değer verir ve onlara yol gösterir.
  • Blog ve destek sayfaları var mı ?
  • Hangi bilgileri paylaşmışlar inceleyin.
  • Aralarında ihtiyaç duyduğunuz bilgiler var mı yok mu bir bakın.
  • Bilgi paylaşmayı seven firmaları üst sıralarda değerlendirin. Çünkü sizinle çalışırken de bilgi paylaşımı konusunda aynı şekilde davranacaklardır.
3) Deneyimlerini öğrenmeye çalışın
İnternet 1998 yılında Türkiye’de ve tüm dünyada yayıldı. Özellikle son on yılda bu alanda çok hızlı bir değişim yaşandı.
Eğer ciddi bir projeniz varsa birkaç senelik iş tecrübesi yeterli olmayacaktır.
  • Eskiden bugüne neler yaptıklarına bir bakın.
  • Web sitelerini hangi tarihte açmışlar. Ne kadar süredir çalışıyorlar ?
Uzun dönemde ayakta kalan web tasarım firmalarını tercih edebilirsiniz. Çünkü, uzun süredir ayakta olmaları sayesinde probemlere karşı pratik çözümler geliştirmelerinin bir sonucu olabilir.
Deneyimli web tasarım ekipleri, daha hızlı ve daha esnek çözümler üretebilir. Tecrübelerini projenize aktarabilirler.
4) Uzmanlık alanları neler ?
Söz konusu web tasarım olduğunda pek çok uzmanlık alanı var. Ancak en çok arayacağınız uzmanlıklar ve nedenleri şöyle sıralanabilir:
  • Dinamik web tasarım: WordPress gibi içerik yönetim yazılımları ile kolayca değiştirilebilen web tasarımlar. Web sitesinde her bir ufak değişiklik için web tasarım firmasını aramak zorunda kalmak istemezsiniz.
  • Kodlama: HTML, CSS ve Javascript yeterli değil. PHP gibi veritabanı ile birlikte çalışan kodlama yetenekleri zorunlu. Özel yazılım veya WordPress gibi yazılımlara özel ilaveler yaparak projenizi daha kısa sürede tamamlayabilmek.
  • Dijital Pazarlama: Projenizin pazarlamasında size yardımcı olabilecek yetenekler. Daha çok ve daha kaliteli ziyaretçi trafiği bir web sitesinin daima ihtiyacıdır.
  • İçerik Geliştirme: İyi bir firma profili yazabiliyor musunuz ? Ziyaretçilerinizi etkileyecek başlıklar ? Google’daki arama sonuçlarında dikkat çeken metinler ? Web tasarım firmaları arasından belki de en az bulunan yeteneklerden birisi. 
5) Dijital pazarlama veya SEO konularında uzmanlıkları var mı ?
Şirketinizin yapısı ne olursa olsun web sitenizin temel ihtiyaçları değişmez:
  1. Başta arama motorları olmak üzere internette görünür olmak
  2. Web sitenize daha çok ve daha kaliteli ziyaretçi çekmek
  3. Gelen ziyaretçileri müşteriye dönüştürecek içerikler
  4. Web sitenizin sürekli iyileştirilmesi için düzenli analizler
Dijital pazarlama, önemi giderek artan bir konu. Bu konularda uzmanlıkları var mı bir inceleyin. Profesyonel ve etkili dijital pazarlama konusunda temel bilgileri buradan öğrenebilirsiniz.
6) Web Tasarım Teknolojileri
Web tasarım firmasının sunduğu teknolojiler ve faydaları:
  • Açık kaynak kodlama: dilediğiniz zaman dilediğiniz kişiye web sitenizde düzenleme yaptırabilmelisiniz. Özetle, web tasarım firmasına bağımlı kalmak istemezsiniz.
  • Performans: Web siteniz ziyaretçi sayısı artsa da performansındaki kayıp çok az olmalı.
  • Güvenlik: İnternet, hackerların rastgele saldırdığı bir ortam. Güvenilir, test edilmiş yazılımlar bu nedenle çok önemli.
  • Bakım ve Süreklilik: WordPress, Joomla veya Drupal gibi yazılımlar ücretsizdir ve sürekli güncellenir. Yeni özelliklere ve hatalara karşı sürekli konursunuz.
Web sitesi tasarımında teknoloji seçimi önemli. Başka bir web hosting şirketine geçmek istediğinizde size sorun yaratmamalı.
Benim önerim: Linux hosting hizmetini tercih edin.
Linux hosting (barındırma) hizmetleri windows paketlerine göre daha avantajlı. Sadece bir teknolojiye bağımlı kalmak, size sürekli maliyet olarak geriye dönecektir.
Web tasarım firmaları araştırmanızda size bugün ve ilerde ek maliyet getirmeyecek çözümleri araştırmalısınız.
Bu nedenle Web Tasarım Sitesi olarak yıllar önce Linux platformunu kullanıyoruz.
PHP ile kodlama ve MySQL veritabanı en popüler uygulamalar.
Linux hosting paketleri daha düşük maliyetli, daha kolay yönetilen ve daha performanslı olduğu için tercih ediyoruz.
7) Hosting ve Alan Adı Tescili
Buradaki sormanız gereken soru: Web tasarım firması hosting konusunda size nasıl yardımcı oluyor ?
a) Alan adı tescili ve hosting hizmeti veren firmalar
Web Tasarım Sitesi olarak biz de pek çok müşterimize paket danışmanlık hizmeti veriyoruz. Web tasarımı, hosting ve alan adı tescilini tek noktadan takip, pek çok sorunun oluşmasını engelliyor. Müşteri için ideal olan, bu tür konularda çok uğraşmamak oluyor.
b) Müşteri alan adı tescili ve hosting hizmetini kendisi temin ediyor
Bu seçenek müşterinin kendi işini kendisi görmesi açısından avantajlı. Alan adı tescili ve web hosting paketi açmak çok karmaşık
işlemler değil. Web tasarım firması dikkat etmeniz gereken teknik detayları verebilir. Burada alan adı ücretlerinin ve hosting ödemesini unutmamak size düşüyor.
Hosting konusunda dikkat etmeniz gereken:
Web sitesinin yavaşlık, kesinti gibi problemler ile bir önceki adımda bahsettiğim bakım ve destek talepleri konusundaki yaklaşımını öğrenmelisiniz.
8) Responsive Tasarım veya Mobil Web Tasarımı
Responsive, türkçesiyle duyarlı web tasarımı, akıllı telefon, tablet veya netbook gibi küçük ekranlı cihazlardan, büyük ekranlara uyum gösteren tasarım anlamına geliyor.
İnternet trafiğinin çoğunluğunu akıllı telefonlar oluşturuyor. Mobil cihazlarla web siteniz uyumlu olmazsa, insanlar sitenizi kullanmayı bırakırlar. Bunu istemezsiniz.
Bu nedenle web tasarım firmaları araştırmanızda bu özelliğe destek verileceğinden emin olun.
9) İçerik yönetim yazılımı
İçerik yönetim yazılımı, web sitenizdeki bilgileri kodlama bilgisi olmadan, kolayca değiştirmenizi sağlar.
İlk akla gelen değişikler:
  • Web sitenizdeki metin veya fotoğraflar
  • İletişim adresiniz, telefon numaranız veya eposta adresiniz
  • Yeni çıkardığınız ürünlerin eklenmesi veya mevcut ürün sayfalarının güncellenmesi
  • Kurumunuzun kazandığı ödül veya sertifika
  • Firmanız hakkında değişen önemli bilgiler vs
Liste uzar gider. Herhangi zamanda ihtiyaç duyacağınız bu değişikliği kolay ve hızlı yapmak isteyeceksiniz.
Web sitenizi yönetmeye yarayan pek çok yazılım buluyor.
İşte ilk akla gelenler:
  • WordPress
  • Joomla
  • Drupal
Özel içerik yönetim yazılımı
Benim önerim: özel yazılımı gerekli olmadıkça tavsiye etmiyorum.
Özel yazlım, son derece zahmetli ve pahalı bir iştir. Daha sonra da size pahalıya mal olmaya devam eder.  Özel yazılımın desteklenmesi, sürekli güncel tutulması büyük maliyetler ve zorluklar oluşturur.
Web Tasarım Sitesinde WordPress kullanıyoruz.  Şu anda okumakta olduğunuz sayfa WordPress ile çalışıyor. Hemen her ihtiyacı karşılayacak özelliklere sahip ve dünyanın en çok kullanılan içerik yönetim yazılımı. Müşterilerimizin özel talepleri olursa WordPress teması veya eklentisi kodlayabiliyoruz.
10) Web Tasarım Firmalarının Çalışma Yetenekleri
Bir web tasarım şirketi için müşterileri ile olan iletişimi, projeleri yönetme ve problemleri çözme biçimi bana göre en önemli noktayı oluşturuyor.
Web Tasarım Sitesi olarak yıllardır karşılaştığımız müşteri şikayetleri, en başta konuşulması gereken konuları atlayarak problemi daha baştan yaratmaya sebep olan davranışlar oluşturuyor.
Web tasarım firmaları araştırmanızda projenizi başarıyla tamamlayacak aşağıdaki yetenekleri aramalısınız:
1) Şirketinizin hedeflerini, ürünlerinizi veya fikirlerinizi iyi anlamalı: Bir web sitesi tasarım projesinin hedefi, ziyaretçilerinize sizin hakkınızda etkileyici görünüm vermek değildir.
Öncelikli görevi; ziyaretçilere ilk başta hedeflediğiniz doğrultuda yönlendirmesidir.
Bu nedenle web tasarım şirketi, sizi ve fikirlerinizi iyi anlamalıdır.

 

2) Sorularınıza açık ve net cevaplar vermeli: Projenizin başarılı olması için hangi fikirlerin çalışacağını, hangilerinin iş yapmadığını iyi bilmek gerekir.
Biz WTS olarak müşterilerimize fikirlerini olgunlaştırmaları için tavsiyelerde bulunuyoruz.
Deneme yanılma metotları ile ilerlemeye çalışmak, zaman ve emek kaybından başka bir şey getirmiyor.
3) İçerik konusunda yardım etmeli: Daha önce de söylediğim gibi içerik sizin temel probleminiz. Dünyadaki tüm başarılı sitelerin ziyaretçileri mıknatıs gibi kendilerine seçmelerinin nedeni içeriktir. Sosyal medya kanalları, arama motorları, haber portalları… İster şirket olun, isterse bireysel girişimci.
Sizin sitenize de ziyaretçileriniz içerik için geleceklerdir. Bu içeriklerin nasıl olacağını iyi düşünmelisiniz.
Web tasarım firması da size bu konuda yardımcı olabilir. Uzun yıllardır içerik geliştirme konusuyla ilgileniyorum. İçerik pazarlaması ziyaretçilerinizi müşteri olmaya ikna edecek pazarlama metotlarının en başında geliyor.

 

4) Etkili, hızlı ve kalıcı bir iletişim kurmalı:  WTS’de başlıca iletişim tercihimiz eposta. Yazılı iletişim, detayları gözden kaçırmamıza engel oluyor.
Bir diğer önemli nokta: eposta mesajlarınıza yanıt süreleri nasıl ?

 

Neden telefon değil ?
Telefon acil durumlar için olmalı düşüncesindeyiz. Elbette projenizi konuşmak gerekli. Ancak projenin büyüklüğü, zamana yayılması ve hatta birden fazla kişinin bilgisiyle yürüyen bir iş olduğunu düşünürseniz eposta en etkili iletişim yoludur. Mesele eposta olunca da cevaplar için günlerce beklememelisiniz.

 

5)Uzaktan çalışma yetenekleri:
Uzaktan web sitesi projesi geliştirmek kimine göre zor bir iştir. Yüz yüze yapılan iletişim olmadığı için sorunlar yaşanabileceği düşünülür. Oysa, web tasarım firması eğer disiplinli ve dikkatli çalışırsa, uzaktan çalışmak kayıp değil kazançtır.

 

İşte uzaktan çalışmanın faydaları:
  • Yer ve mekan problemi: İstanbul gibi büyük şehirlerde ulaşım zordur. Web tasarım firması ile yüz yüze görüşmek her seferinde zaman kaybına neden olur. Uzaktan çalışmak bu tür sorunları ortadan kaldırır.
  • Randevu kolaylığı: Telefon, Google Hangout veya Skype ile görüntülü bir toplantı, günümüz iş dünyasında çok zor bir iş değil. Günün size uygun saatinde randevulaşabilirsiniz.
  • Odaklanma: Ulaşım ve görüşme meselelerini ortadan kaldırdığınızda, kısa online randevu veya eposta yazışması ile sadece projenize odaklanma şansına kavuşursunuz.

 

Web tasarım firması ile aynı şehirde bile olsanız, uzaktan çalışma disiplinine sahip olduğuna bir bakın. Faydası çok olacaktır.
11) Web Tasarım Fiyatları konusundaki politikaları
Şunu kabul edelim web sitesi tasarım hizmeti bilgiye dayalı ve maliyetli bir iştir. Size “pahalısı iyi, ucuzu kötüdür” demeyeceğim.
Bundan önce mutlaka bir alışveriş deneyiminiz olmuştur. Kalite/fiyat bağlantısını bilirsiniz.
Ancak, fiyatlar konusunda bazı ipuçları vererek değerlendirmenize katkıda bulunmak istiyorum.
Soru 1: Tasarım hizmetlerini nasıl fiyatlandırıyorlar ?
Burada iki türlü fiyatlandırma metodu bulunuyor:
1) Paket
Web tasarım firması, hazırladığı paketlere birer fiyat veriyor. Belirli özelliklerin bulunduğu tasarım paketleri.
Artısı: Web sitesi sahibi, ödediği ücret karşılığında ne alacağını biliyor.
Eksi: Paket fiyatları kandırıcı olabiliyor. İstediğiniz özelliklerle uyuşmadığında daha sonra ilave maliyetler çıkabiliyor.
2) Proje 
Proje bazında iki türlü fiyat var.
Saatlik: Saat başına ücret vererek, proje ne kadar süreye ihtiyaç duyarsa o kadar çalışılıyor ve maliyet saat ücretine göre oluşuyor.
Artısı: Saatlik ücret ödeyerek istediğiniz kadar web tasarım firmasını çalıştırabilirsiniz. Örneğin projenin bir kısmı için bir firmayı tutabilirsiniz.
Eksisi: Projenin bitişi ile ilgili net bir durum yoksa, proje maliyetleri artarak devam eder.
Projenin tamamı: Bu müşteri için en uygun seçenek.  Eğer müşteri doğru ve detaylı bilgiler verirse, tek bir maliyet belirleniyor.
Artısı: Müşteri, tek bir fiyat karşılığında projenin ne kadar sürede biteceğini bilerek web tasarım firmasını işe alıyor.
Eksisi: Eğer yeterli detay belirlemezse, proje tamamlandığında konuşulmamış yarım işler ortaya çıkabiliyor. Bu noktada, yeniden bir fiyat çalışması yapmak gerekiyor.
Web tasarımı fiyatlandırmasında tek bir yöntem yok.
Ben proje bazında fiyat almanızı öneririm.
Ancak isteklerinizi birinci adımda söylediğim gibi iyi belirleyin. Aksi takdirde web tasarım firmasını bu konuda kendisini sorumlu tutamazsınız.
Soru 2: Web sitenize proje tesliminden sonra da destek olacaklar mı ?
Projeniz eksiksiz tamamlanmış bile olsa, zaman içerisinde bazı ihtiyaçlarınız olabilir. Her durumda aşağıdaki noktaları açıklığa kavuşturmalısınız:
  • Hosting ve alan adı ödemelerini kim yapıyor ?
  • Web sitesinde oluşabilecek teknik sorunlar nasıl çözülecek ?
  • Yedekleme yapılacak mı ?
  • Güvenlik önlemleri neler ?
  • Web sitesine ilaveleri kim yapacak ? Koşullar nelerdir ?
  • Web sitenizin dijital pazarlama ihtiyaçlarını nasıl ve kimden karşılayacaksınız ?
Eğer WordPress gibi bir içerik yönetim yazılımınız varsa, yedekleme ve içerik güncelleme ihtiyaçlarınızı kendiniz çözebilirsiniz.
Özel yazılım veya HTML ile kodlanmış bir web sitesinde güncelleme için kodlama bilgisine ihtiyacınız olacak. Bunu istemezsiniz.
12) Web Tasarım Referansları
Referanslara bakarken dikkat edeceğiniz noktalar:
  • Web tasarım firması sadece bir sektöre mi odaklanmış, yoksa farklı alanlarda proje üretebiliyor mu ? Odaklandıkları alan size uygunsa sorun yok. Diğer türlü beklediğinizi alamayabilirsiniz.
  • Tasarımlar birbirine benzeyen fotokopiler mi,  yoksa farklı ve özgün tasarımlardan mı oluşuyor ?
  • Hangi teknolojileri kullanarak gerçekleştirmişler ?
  • Tasarımın yanında içerik ve mesajlar ziyaretçi ile bağ kurabiliyor mu ?
  • Referanslar güncel mi, yoksa uzun süredir yenilenmemiş mi ?
Sadece kendi sektörünüzdeki tasarımlarla değerlendirme yapmayın. Farklı sektörlerdeki çalışmalarını da inceleyerek, farklı alanlardaki yeteneklerini inceleyin.
13) Uzak durmanız gerekenler (yine de siz bilirsiniz)
Web tasarımı işine başlamak için neredeyse bir bilgisayardan fazlası gerekmiyor.

 

Ancak, başarılı olmak sabır ve emek gerektiriyor.

 

Aşağıdaki türdeki insanlar sizin de karşınıza çıkabilir. Web Tasarım Sitesi’ndeki tecrübelerime göre şu kişilerle çalışmak sadece zarar etmenize neden oluyor:

 

1) Hotmail, Gmail gibi ücretsiz eposta adresi kullananlar: webtasarimsitesi.com gibi bir alan adı almak çok pahalı bir iş değil. Bana göre hotmail gibi ücretsiz mail kullananlar işlerini yeterince önemsemiyorlar. Bir alan adı dahi satın almadan çalışıyorlarsa yarın nerede olacaklarını kim bilebilir ki ?

 

2) Kendisini tanıtan düzenli ve güncel bir web sitesi olmayanlar:  Eğer söküğünü dikmeyen bir terzi görürseniz ne düşünürsünüz ? Onun işine ne kadar güvenirsiniz ? Eğer bir kişi “web tasarımı yapıyorum” diyor, web sitesi “tasarım aşamasında” ise oradan hızla ayrılın.

 

3) Size kopya resim, grafik veya içerikler önerenler: Başka sitelerden aldığı grafik, resim, içerik veya hatta web tasarımı önerenler sadece başınıza bela getirir. İnternet gizli saklı bir yer değil. Web sitenizdeki çalıntı ve lisansı olmayan materyallerden site sahibi olarak doğrudan siz sorumlu olursunuz. Bunu unutmayın.

 

4) Flash, asp (asp.net değil) gibi eski teknolojilerle web sitesi yapanlar: Tecrübelerime göre bu tür kişiler teknolojiyi en az 5 sene geriden takip ediyorlar. İnternet gibi her şeyin hızlı değiştiği bir alanda 5 yıl çok uzun bir süredir.

 

5) Yaptığı işi sergilemeyen, referans ve iletişim bilgilerini vermekten kaçınanlar: Yaptığı projeleri sergilemeyenlerden arkanızı dönüp doğrudan kaçmanız faydalı olur.

 

6) Alan adınızın ve web hosting (barındırma) hizmetlerinizin şifrelerini vermeyenler: Alan adı ile ilgili sıkıntılar çok büyük zararlara yol açabiliyor. Şirketler epostalarını, müşterilerini kaybediyor. Eğer sağdan soldan bu firma ile ilgili müşterilerine şifrelerini vermediklerini duyarsanız, çalışmayın.

 

7) Size sunabildiği tek avantajı “ucuz” web tasarımı olanlar: Yaptıkları işte iyi olmayanlar “ucuz” kelimesinin arkasına saklanırlar. Fiyatı “ucuz” olan bir web sitesi size daha sonra çok daha pahalıya patlayabilir.

 

18 yıllık tecrübemde öğrendiğim şu gerçeği tekrar etmek istiyorum:
Ucuz web sitesi yaptıranlar zamanla daha kaliteli projeleri arıyorlar. Yani önce fiyatına bakarak karar verip, belki aylar sürecek oyalamadan sonra ciddi bir web tasarım firması ile çalışmaya karar veriyorlar. “Ucuz” zaman ve para kaybına sebep olabiliyor.

 

8) Kısa süre önce bu işe girmiş, esas işi başka iş olanlar: Başta da söylediğimiz gibi web tasarım firması kurmak veya serbest çalışan olarak girişmek çok zor değil. Asli işi başka olanlar, her an başka bir işe geçecek olanlar sizi yarı yolda bırakarak başka ufuklara her an yelken açabilir.

 

9) Sözleşme önermeyenler: Yapılan her işin elinizde bir sözleşmesi olması, formalite gibi görünse de her iki tarafın çıkarına bir durumdur. Sözleşme, karşılıklı her iki tarafın yararına düzenlenmeli, işin detaylarını yeterli seviyede barındırmalıdır.

Adım 3:  Sizin için En iyi Web Tasarım Firması Değerlendirmesi

Web tasarım firmaları arasından seçim yapmak için ihtiyacınız olan iki şey var:

1) Projenizin detaylarına hakim olun

Ne istediğinizi, ne kadar süre ve bütçenizin olduğu, hangi hizmetlere ihtiyaç duyduğunuzu ve aradığınız firmalardaki yetenek, referans ve teklifleri ile ne kadar iyi karşılaştırdığınıza bağlı.

2) Uzun vadeli bir iş ortaklığı kurmak üzeresiniz

Web tasarım olarak adlandırılan iş artık sadece web sitesi tasarımından ibaret değil.
Türkiye’de web tasarım firmaları, alan adı ve web hosting başta olmak üzere müşterilerinin pek çok sorunuyla ilgileniyorlar.
 Web Tasarım Sitesi’nde, müşterilerimize danışmanlık hizmetleri veriyoruz. Takıldıkları sorunları bize danışmaktan çekinmiyorlar. Bu nedenle uzun vadeli bir iş ilişkimiz oluyor.
Bu nedenle…
Bir seferlik iş gözüyle bakmayın, uzun vadeli düşünün.
