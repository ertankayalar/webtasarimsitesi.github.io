WordPress Web Tasarım Seçeneklerinizi Keşfedin

Yazan Ertan Kayalar | Güncelleme Şub 3, 2019 | Web Sitesi | 0 yorum

Herkes web sitesinin daha güzel görünmesini ister. İşin içine girince yönetimin de daha kolay olmasını, maliyetin daha düşük olmasını vs. vs. Etkileyici görünüm için WordPress web tasarım çalışmaları yaparken gözönünde bulundurmanız gereken noktalar var. Web sitesindeki amacınız ne olursa olsun, doğru bilgilere sahip olarak hedeflerinize ulaşmak için daha çok şansınız olacak.

WordPress’in en sevdiğim özellikleriden birisi medya yönetimidir. Resim yüklemek gibi çok sık yapılan bir işi ustalıkla yapmanızı sağlayarak kolaylaştırır. Bütün resimleriniz içinde arama yapmak, boyutlarını değiştirmek gibi temel işlemleri rahatlıkla yapabilirsiniz.

Neden WordPress Web Tasarım Avantajlı ?

WordPress ile sitenizi tasarlatmak için pek çok nedeniniz var.

  • Ücretli premium temalarla web siteniz profesyonel görünüme sahip olabilir
  • Temalar wordpress yazılımından ayrı hazırlanır. Ceket gibi kolayca değiştirebilirsiniz.
  • Tasarımınızı çocuk temalarla özelleştirebilirsiniz.
  • PHP açık kodlu yapıda olduklarından kolayca değiştirilebilirler.
  • Çok dil desteği ile web sitenizin birden fazla dilde doğru görünmesi sağlanır.
  • WordPress eklentileri ile web siteniz ve tasarımınız zenginleşir.

Eğer WordPress konusunda yeniyseniz neden wordpress’i kullanmanız gerektiğini anlatan yazımı okuyarak öğrenebilirsiniz.

WordPress teması ile web sitenizin görünümünü değiştirin

WP’nin gücü kusursuz çalışan özelleştirme yapısından gelmektedir. Sitenizin nasıl olmasını istiyoranız onun için çalışırken, aynı zamanda çekirdek sisteme zarar vermeden bunları yapmanız mümkündür. Bu yapı sayesinde web sitenize görsellik ve fonksiyonellik kazandırmak çok kolay hale gelmektedir.

WordPress’i iki temel şekilde özelleştirebilirsiniz. Birincisi, eklenti (plugin) adı verilen ekleme yazılımlar ile sitenize yeni fonkisyonlar ekleme şansınız olur.

Nextgen Gallery eklentisi ile web sitenizde fotoğraf galerileri oluşturabilirsiniz.

Bir diğer eklenti Contact Form 7 ile web sitenize iletişim formu, iş başvuru formu veya basit sipariş formu ekleyebilirsiniz.

Wordress Teması (WordPress Theme) Nedir?

Web sitenizi WordPress ile özelleştirmenin diğer yönü ise Temalardır. İngilizcesi Template olan temalar ile web siteniz tamamen farklı bir görüntüye kavuşabilir.

WordPress içinde hazır temalarla gelmektedir. Bu temalalar basit bir blog veya web sitesi açabilirsiniz.

WordPress içerisinden internet üzerindeki temalardan yükleyebilirsiniz.

Web tasarımınızı size özel yapmak için çocuk tema kullanın

Problem şudur; bir wordpress temasını satın alır ve PHP kodlarını düzenleyebilirsiniz. Ancak satın aldığınız temanın yeni bir sürümü çıktığında güncelleme yaparsanız yaptığınız düzenlemeler iptal olur. Ya eski, belki de güvenlik açığı bulunan, size özel temayı kullanacaksınız. Ya da temayı güncelleyip özelleştirmeleri baştan yapacaksınız.

Bu sorunu basit bir şekilde çözmek mümkündür. Child Theme, çocuk tema. Temanın aslına dokunmadan size özel düzenlemeler çocuk tema üzerinde yapılır ve ana tema rahatlıkla güncellenir. Hem de size özel yapılan düzenlemelere bir şey olmaz.

Eğer WordPress web tasarım yaptıracaksanız özelleştirmelerin çocuk tema üzerinde yapıldığından emin olun.

WordPress teması seçerken dikkatli olun

Öncelike dikkat etmeniz gereken nokta; yasal olmayan yollarla kuracağınız temalar büyük risk taşır. Güvenmediğiniz web sitelerinden tema indirip sitenize kurmayın. Web sitenize zararlı yazılımları da birlikte kurma tehlikesi ile karşı karşıya kalırsınız. Ayrıca lisans sorunu nedeniyle başınız ağrıyabilir. Lisanslı WordPress temalarından teknik destek ve güncelleme hizmetinin avantalarını da ekleyelim.

Premium temaları seçerken dikkat etmeniz gereken noktalar

WordPress web tasarım çalışmanızın en önemli noktası projeniz için doğru temayı seçmektir. Bu noktada doğru temayı seçmeniz için size basit bilgiler:

  • Güncel WordPress sürümü ile çalıştığından emin olun.
  • Mobil cihazlarla uyumlu bir tasarımı tercih edin.
  • Türkçe ve çok dil destekli çalıştığını kontrol edin.
  • WPML eklentisi ile uyumlu olması daha iyi çok dil desteği kazandırır.
  • Kurumsal, eğitim, magazin, eticaret gibi farklı amaçlar için hazırlanan temaları işinize uygun olanı seçmeye çalışın.
  • Eğer eticaret yapacaksanız, Woocommerce eklentisi ile uyumluluk işinize yarayabilir.
  • Yorumları ve değerlendirmeleri okuyun.

Themeforest, WordPress temasının hangi sürümle çalıştığını, ne tür dosyalar teslim edildiğini ayrıntılı şekilde gösterir.
